# "Определение возраста покупателей"

## Данные

Исходными данными являлись фотографии людей взятые с сайта
  
## Задача

Сетевой супермаркет внедряет систему компьютерного зрения для обработки фотографий покупателей. Фотофиксация в прикассовой зоне поможет определять возраст клиентов, чтобы анализировать покупки и предлагать товары, которые могут заинтересовать покупателей этой возрастной группы и контролировать добросовестность кассиров при продаже алкоголя. Наша задача построить модель, которая по фотографии определит приблизительный возраст человека. 

## Навыки и инструменты
*Python*, *Pandas*, *TensorFlow*, *Keras*

## Краткое описание выполненного проекта
Провела исследовательский анализ данных: определила размер выборки, построила график распределения возраста в имеющейся выборке.
С помощью Keras обучены модели. Качество моделей оценивалось с помощью функции потерь МАЕ (модель обучалась в GPU-тренажере)
